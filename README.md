# Abubeker Mubarek - Portfolio

A modern, interactive 3D portfolio website showcasing my work as a Senior Software Developer.

## Project info

This is my personal portfolio website featuring:
- Interactive 3D visuals with Three.js
- Responsive design
- Professional resume/CV section
- Project showcase
- Contact information

## Technologies Used

This project is built with:

- **Vite** - Fast build tool
- **TypeScript** - Type-safe JavaScript
- **React** - UI framework
- **shadcn-ui** - Component library
- **Tailwind CSS** - Utility-first CSS
- **Three.js / React Three Fiber** - 3D graphics
- **React Three Drei** - 3D helpers

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd mubarek-3d-architect

# Install dependencies
npm i

# Start the development server
npm run dev
```

### Build for Production

```sh
npm run build
```
